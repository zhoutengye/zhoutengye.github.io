买房收益的计算

| 自由之路 理财思考
| 阅读

以前其实也没自己算过贷款利率这些，最近因为在摇号， 就了解了下，并稍微计算了下收益率。 用的是jupyter notebook写了点小程序计算， 也是为了方便改参数看不同结果，已经后面还可以画点图。 代码部分可以忽略不计，公式也可以不看，直接看第二部分的文字即可。

一些计算用的程序

首先简单写一些程序，用于后续的对比。

等额本金还款

总贷款额$f$，贷款年利率为$r$，贷款年份为$y$，则总还款期数为$12y$，每期归还本金$a$为 $$a(n)=\frac{f}{12y}$$ 对于第$n$期来说，此时欠银行的本金为 $$t(n)=f-na(n)$$ 第$n$月应每月应还利息$b$以剩余本金 $t(n)$进行计算: $$b(n)=\frac{tr}{12}$$ 每期应还总金额为 $$c(n) = a(n)+b(n) $$ 前$n$期总付款金额为 $$s(n)=\Sigma_{1}^{n}c(n)$$ 前$n$期总利息为 $$bb(n)=\Sigma_{1}^{n}b(n)$$

import numpy as np
def interest1(f,r,y,n):
    s = 0
    an = np.zeros(n)
    bn = np.zeros(n)
    cn = np.zeros(n)
    s=0
    for i in range(n):
        a = f / y / 12.0
        t = f - i * a
        b = t * r / 12.0
        c = a + b
        s = s + c
        an[i] = a
        bn[i] = b
        cn[i] = c
    bb = np.sum(bn)
    return an, bn, cn, s, t, bb

等额本息还款

同样总贷款额$f$，贷款年利率为$r$，贷款年份为$y$，则总还款期数为$12y$， 每期的还款总额$c$为固定值， 则每一期的递推公式为 $$t(1) = (1+\frac{r}{12})f - c$$ $$t(2) = (1+\frac{r}{12})t(1) - c $$ $$… $$ $$t(12y-1) = (1+\frac{r}{12})t(12y-2) - c $$ $$t(12y) = (1+\frac{r}{12})t(12y-1) - c = 0$$ 第$n$期欠银行的钱为 $$ t(n) = f(1+\frac{r}{12}）^n - c(1+\frac{r}{12})^{n-1} - … - c(1+\frac{r}{12})^{1} - c $$ 使用等比数列求和公式可得 $$ t(n) = f(1+\frac{r}{12}）^n - c((1+\frac{r}{12})^{n}-1）/n) $$ 当$n=12y$时,还清所有贷款， $$ t(12y) = f(1+\frac{r}{12}）^{12y} - c((1+\frac{r}{12})^{12y}-1）/12 / y)=0 $$ 由此可以得到$c$的取值 $$ c = \frac{f(1+\frac{r}{12}）^{12y}}{((1+\frac{r}{12})^{12y}-1）/12 / y)} $$ 每期归还利息为 $$a(n) = \frac{r}{12}t(n-1)$$ 每期归本金为 $$b(n) = c - a(n)$$

def interest2(f,r,y,n):
    c = f * (1+r/12.0)**(12.0*y)/( ((1+r/12.0)**(12.0*y)-1)/r*12 )
    c = (r/12.0/(1-(1+r/12.0)**(-y*12)))*f
    an = np.zeros(n)
    bn = np.zeros(n)
    cn = np.zeros(n)
    t=f
    s = 0
    for i in range(n):
        cn[i] = c
        bn[i] = t * r / 12.0
        an[i] = cn[i] - bn[i]
        t = t - an[i]
        s += cn[i]
    bb = np.sum(bn)
    return an, bn, cn, s, t, bb

买房收益计算

假设房价为$A$，买房首付$A_1$为$0.3A$，贷款年份为$y$，第$n$个月卖出，卖出房价为B， 再考虑一个房屋出租的收益，假设每个月$u$。 则总收益S(n)为 $$S=B-A-bb(n) + u\times n$$

# 等额本金option=1， 等额本息option=2
def house_profile(A,B,r,y,n,u,option):
    f = 0.7 * A
    if option == 1:
        an, bn, cn, s, t, bb = interest1(f,r,y,n)
    elif option == 2:
        an, bn, cn, s, t, bb = interest2(f,r,y,n)
    S = B - A - bb + u * n
    D = 0.3 * A + np.sum(cn)
    return D,S

定投模型

假设有一个年化收益百分之$k$的项目（例如标普500指数），则如果上述的买房首付+月供模式可以等价为一个定投模型。由于没有买房，所以无法获得租金。 由此以同样的首付和每月投入可以获得的资金总量公式为一个递推公式: $$M(1) = 0.3(1+\frac{k}{12})A $$ $$M(2) = (1+\frac{k}{12})M(1)+c(1) $$ $$M(3) = (1+\frac{k}{12})M(2)+c(2) $$ $$… $$ $$M(n) = (1+\frac{k}{12})M(n-1)+c(n-1) $$ 到第 $n$ 个月收益为 $$S(n) = M(n) - 0.3A - \Sigma_{1}^{n}c(i) $$

def automatic_profile(A,k,r,y,n,option):
    f = 0.7 * A
    if option == 1:
        an, bn, cn, s, t, bb = interest1(f,r,y,n)
    elif option == 2:
        an, bn, cn, s, t, bb = interest2(f,r,y,n)
    kk = k/12.0
    ms = np.zeros(n)
    ms[0] = 0.3 * (1+kk) * A
    for i in range(n-1):
        ms[i+1] = (1+kk) * ms[i] + cn[i]
    S = ms[-1] - 0.3 * A - np.sum(cn)
    D = 0.3 * A + np.sum(cn)
    return D, S

实际买房收益对比

看中了一个叫智障的楼盘（智障比较容易打字，准确的字找起来麻烦）。政府限购，如果可以摇到的话，价格大概是250w左右。贷款肯定选最长的30年了，年利率看起来似乎是5.9%，5年限售，考级到交房时间，大概就认为6年吧。目前房价2万6左右，周边二手房3.5-4万。

要看收益比的话，首先要对卖出房子的价格进行预估。95平的房子250万左右，我给出6年后麦450万的预估吧，盈利应为200万的房屋增值，减去6年额外付出的利息，加上预估房租收入，得到总收益。同时也需要考虑6年的总资金投入。

在此把函数的输入参数设定为：

买入房价$A$: 250

卖出房价$B$: 450

贷款年利率$r$: 0.059

贷款年数$y$: 30

月租金$u$: 0.35

贷款选项option: 1:等额本金， 2：等额本息。

一些小结：

  • 对比来看，等额本金比等额本息多了15万的资金投入，获得了多了将近4万的收益。再考虑到等额本金前期还款较多，因此等额本息优于等额本金。
  • 即使首付的75万是借来的，需要以年化百分之10付利息，那也是少了45万收益，总体收益依然可观。 另外，可以去对比下用同样的资金去进行复利投资。 假设所有条件不变，没有房子收入，年化收益率百分之8(例如标普500指数)， 可以得到收益为。

那么，6年后房价低到什么程度，收益率会差于年化百分之8定投呢？答案是房价仅为360万左右，即相当于周边二手房房价没有变化。

小结

似乎把jupyter下载成markdown之后在这个模板里看起来效果不好。 我简单总结下： 250万成本买入的房子，采用等额本息还款，中途出租，6年后假如450万卖， 大约可以收益165万左右。首付投入75万，后续还需要投入77万左右，每个月投入1.05万左右。 作为对比，同样的钱去进行年化百分之8的投资，就算叠加复利，也就只有70万左右收益。
