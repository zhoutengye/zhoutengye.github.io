雷公交易系统（一）：双均线指数TradingView代码

2021年1月17日
| 自由之路 理财思考
| 阅读

算是真正开始学习雷公的知识体系了吧。 雷公的免费资源里讲理念， 具体实现要通过买他的会员得到。 399美元一年的会员费并没有到不可以接受的程度， 只是觉得，公开的内容很多， 很值得学习， 等学习完了再决定要不要去开会员。

此次参考视频如下：

其实很容易实现，TradingView里面都有现成的完成度很高的代码， 根据策略整合一下十分容易。 两个指标也很简单，也不仔细分析了， 后期学习的时候再结合其他东西一起写。 作为初步练手，还是直接用了雷公的20天。60天，120天三根均线， 当然这个可以在代码或者指标设置里更改。 我用的暗色主题， MA用的是红色，蓝色，紫色； EMA用的是白色，黄色，绿色。

可惜TradingView的外接版本不支持自定义指数， 也就无法在这里把做双均线系统了， 只能直接把代码复制到TradingView里。 代码如下：

//@version=4
study(title="双均线交易系统", shorttitle="雷公", overlay=true, resolution="")
len1 = input(20, minval=1, title="均线1")
len2 = input(60, minval=1, title="均线2")
len3 = input(120, minval=1, title="均线3")
src = input(close, title="Source")
// MA
offset = input(title="Offset", type=input.integer, defval=0, minval=-500, maxval=500)
ma1 = sma(src, len1)
ma2 = sma(src, len2)
ma3 = sma(src, len3)
plot(ma1, title="MA20", color=color.red, offset=offset)
plot(ma2, title="MA 60", color=color.blue, offset=offset)
plot(ma3, title="MA 120", color=color.purple, offset=offset)
// EMA
ema1 = ema(src, len1)
ema2 = ema(src, len2)
ema3 = ema(src, len3)
plot(ema1, title="EMA 20", color=color.white, offset=offset)
plot(ema2, title="EMA 60", color=color.yellow, offset=offset)
plot(ema3, title="EMA 120", color=color.lime, offset=offset)
